# Acknowledgement

Mocha is built on top of a lot of great open-source work. Thank you to the maintainers of:

- **[DiscordGo](https://github.com/bwmarrin/discordgo)** — the Discord API binding powering the bot.
- **[Echo](https://echo.labstack.com/)** — the HTTP framework behind the backend API.
- **[GORM](https://gorm.io/)** — the ORM used for persistence.
- **[Asynq](https://github.com/hibiken/asynq)** — the Redis-backed queue used for background work.
- **[Meilisearch](https://www.meilisearch.com/)** — powering [room discovery](/web/discover).
- **[Redis](https://redis.io/)** — caching and pub/sub glue.
- **[Next.js](https://nextjs.org/)** and **[Chakra UI](https://chakra-ui.com/)** — the web app at [mocha-bot.xyz](https://mocha-bot.xyz).
- **[Docusaurus](https://docusaurus.io/)** — what you are reading right now.

And a huge thank-you to the community in the [support server](https://discord.mocha-bot.xyz/) for every bug report, feature request, and conversation that shapes the bot.
