Discover Rooms

The public room directory lives at mocha-bot.xyz/search. It is the easiest way to find public rooms to join without needing an invitation code.

What you can do​

  • Browse all public rooms in an infinite-scrolling catalog.
  • Search rooms by name (results are debounced as you type).
  • Filter the list by:
    • Star rating
    • Number of servers joined (min / max)
    • Tags / categories
  • Tap a room card to open the room page for details and a join code.

On mobile the filter panel is a drawer; on desktop it sits in the sidebar.

What you see on a card​

Each card shows the room's name, current average rating, the number of servers currently joined, and any tags. Rooms behind a password still appear in search but cannot be joined without the password.
